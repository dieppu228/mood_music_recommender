\chapter{Giới thiệu}

\section{Đề tài}

Project \textit{Music Mood Recommender} xây dựng pipeline thu thập metadata bài hát từ Spotify, tiền xử lý và phân tích dữ liệu, trích chọn đặc trưng văn bản/âm thanh, rồi chuẩn bị corpus phục vụ retrieval ngữ nghĩa. Hệ thống agent gợi ý (LangGraph + MCP) triển khai ở phiên bản V1; báo cáo tập trung phần khai phá dữ liệu.

\section{Mục tiêu}

\begin{enumerate}[label=\arabic*.]
  \item Thu thập metadata Spotify theo truy vấn mood, đạt 10.197 bài.
  \item Khám phá dữ liệu: phân phối mood, nguồn nhãn, giá trị thiếu, chất lượng văn bản.
  \item Trích xuất 54 chiều đặc trưng âm thanh từ preview Deezer 30 giây.
  \item Đo rò rỉ nhãn (label leakage) trên đặc trưng văn bản.
  \item Chuẩn bị corpus 2.261 bài và embedding cache 768 chiều (Gemini).
\end{enumerate}

\section{Phạm vi}

\textbf{Trong phạm vi:} crawl Spotify hybrid; tiền xử lý; EDA; audio features (librosa); corpus JSONL; embedding; mô tả agent V1.

\textbf{Ngoài phạm vi V1:} pipeline lyrics; vector database Qdrant; đặc trưng âm thanh trong scoring retrieval; huấn luyện deep learning quy mô lớn.

Spotify Developer Mode không trả lyrics, valence/energy, thể loại (genres), URL preview --- hạn chế được ghi nhận ở Bước 3 khám phá dữ liệu.
