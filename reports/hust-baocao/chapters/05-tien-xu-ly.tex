\section{Làm sạch văn bản}

Bước 7 thực hiện:
\begin{itemize}
  \item Chuẩn hóa giá trị thiếu thành chuỗi rỗng (tránh token ``nan'').
  \item \textbf{Mô tả metadata} = tên bài + nghệ sĩ + album --- dùng huấn luyện và suy luận.
  \item \textbf{Ngữ cảnh crawl} = truy vấn + tag --- chỉ dùng phân tích, không huấn luyện classifier.
  \item Token hóa mô tả metadata, bỏ stopword, loại token trùng.
  \item Bỏ tên nguồn generic (``Liked Songs'') khỏi feature.
\end{itemize}

\section{Corpus và embedding}

Corpus JSONL gồm 2.261 bài. Trường tóm tắt metadata ghép tiêu đề, album, năm, mood, tag. Trường lời bài để trống; thể loại rỗng trên crawl hiện tại. Tập mẫu kiểm thử: 30 bài (5/mood, seed=42).

Ma trận embedding $2261 \times 768$, kiểm tra tính toàn vẹn corpus bằng hash SHA-256.
